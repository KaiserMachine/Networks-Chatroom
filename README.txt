Chatroom - Made by Omar and William.

NOTE: This is just a toy example for learning about networks, there is no encryption on this chatroom.

STEPS TO RUN:
1: Open Server.java and run it.
2: Open ChatScreen.java in a terminal and type the following command:
ignoring brackets: java ChatScreen localhost [user]
3: Repeat process to add as many users as desired.
4: Exit when done.
